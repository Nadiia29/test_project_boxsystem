✨ Project Overview

Elite Auto Gallery is an immersive animated showroom experience featuring:

🌓 Dual-phase environment (exterior → interior reveal)

🎬 GSAP-powered door animation with dynamic mouse-tracking

🎥 Cinematic looping video backdrop

🌟 Sparkling particle effect built manually with DOM + GSAP

🚗 Interactive car collection

 - smooth scroll-in animations

 - click-activated fullscreen overlay

 - ESC-to-close, fade/scale animations

🔧 Clean component structure optimized for reusability

📱 Fully responsive layout

The project focuses on high-quality motion, depth, and atmosphere, creating a luxury brand feel.

🔥 Key Features
🎯 Hero Entrance Animation

Sliding doors synchronized via GSAP timeline

Mouse-tracking background parallax

Fade-out hero content

Transition to interior environment & video fade-in

🎥 Cinematic Video Background

 - Autoplay, muted, loop

 - Smooth opacity reveal triggered by scroll

🌟 Custom Sparkles Component

 - 50 randomly generated sparkles

 - Looping, randomized fade animations

 - Efficient cleanup using GSAP context

🚗 Interactive Collection Grid

 - Alternating left/right slide-in animations

 - Click to open a detailed overlay

 - ScrollTrigger for staggered motion
